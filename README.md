# PrimerProyectoReact
Primer proyecto utilizando ReactJS. Aprendiendo a utilizar esta herramienta

En este proyecto se muestra la estructura básica de un proyecto en React, el cual está conformado por elementos y componentes.
Se puede notar la forma en la que un componente se conecta al HTML y se logra visualizar por medio del método render, que se aplica en el
DOM propio de React.
